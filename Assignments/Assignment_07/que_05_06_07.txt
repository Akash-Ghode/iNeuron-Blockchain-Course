 5. What string method can be used to convert string into array ?

 ==> "split" method can be used to convert string to array.

 e.g.  "I am Akash".split(" ");
       Output: ['I', 'am', 'Akash']


6. What string method can be used to check the occurrence of a specified text in a string?

==> includes method can be used to check the occurrence of a specified text in a string.

e.g.  "I am Akash".includes("m Aka")
       Output: true
